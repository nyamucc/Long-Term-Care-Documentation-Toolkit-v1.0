# Long-Term-Care-Documentation-Toolkit-v1.0
A workflow automation toolkit that improves accuracy, compliance, and efficiency in long‑term care medication documentation.

## Overview

The Long‑Term Care Documentation Toolkit v1.0 is a comprehensive, automated Google Sheets system designed to reduce documentation errors, streamline caregiver workflow, and support audit‑ready monthly records across multiple residents.
This toolkit includes:
- A dynamic caregiver dropdown powered by a Named Range
- A monthly MAR (Medication Administration Record)
- A controlled‑substance (narcotics) log with caregiver witness
- A vitals and behaviour monitoring sheet
- A hospital visit tracker
- An incident reporting log
- A medication changes log
- A resident profile sheet
- A workflow that supports monthly continuity and long‑term scalability

## Key Features

- Dynamic Caregiver Dropdowns.
All caregiver selections across all sheets are linked to a single Named Range

- Error‑Proof Data Validation.
Each caregiver column uses one unified rule applied to the entire column, ensuring consistent documentation and preventing broken validation.

- Monthly Duplication Workflow.
Each month, the resident sheet is duplicated to preserve historical records and maintain compliance.
This ensures:
No overwritten data
Clean monthly logs
Easy auditing
Consistent caregiver workflow

- Scalable Multi‑Resident Design.
The system supports:
- Multiple residents
- Multiple caregivers
- Multiple months
- Multiple facilities
All without breaking formulas or requiring manual reconfiguration.
